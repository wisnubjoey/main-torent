if you want to run the server, you need to run the following command:
```bash
composer run dev
```
if using mcp only read query that is allowed without asking me to confirm, update, delete and any other except read query must ask for permission to confirm.

If you want to create a migration, you need to run the following command:
```bash
php artisan make:migration (table name)
```

This is production project, make sure to make everything is working as expected before deploying to production.
